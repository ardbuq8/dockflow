## 用起来有多简单

三步，真的就三步：

1. 把你要用的应用打开，排好位置
2. 在 DockFlow 菜单栏图标里点"保存当前 Dock"，给它起个名字（比如"写代码"）
3. 之后想用这套布局，点一下就切换过去

也可以设置快捷键，在不同方案之间直接键盘跳转，完全不用动鼠标。

DockFlow 住在菜单栏里，不占 Dock 位置，不需要任何额外权限，轻量到你几乎感觉不到它存在——直到你需要切换的那一秒。

## 功能点逐一看看

**创建多套图标方案**
可以设置任意数量的 Dock 方案，每套方案里可以放 App、文件夹、文件，甚至网址链接。想在 Dock 里加个"打开公司内网"的快捷方式？完全可以。

**添加空白间隔**
支持大/小两种空白间隔符，可以把 Dock 图标分组排列，看起来整洁很多。这个功能说起来小，但用上之后会发现挺上头的。

**配置导出/导入**
所有个性化方案都可以导出备份，换新 Mac 或者重装系统之后直接导入，不用重新设置一遍。

**自动关闭不相关应用**
切换方案时，可以选择自动关掉不在当前方案里的应用。这个功能对于需要保持专注的场景很有用，尤其是进入"深度工作"模式时，让视野一下子清净不少。

**支持 Apple 快捷指令 & CLI**
喜欢折腾自动化的用户，可以通过 Apple Shortcuts 或者命令行来触发 Dock 切换，配合 Focus 模式用起来更丝滑。
